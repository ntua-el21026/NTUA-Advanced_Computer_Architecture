# Assignment Materials

This directory stores the first exercise handout and reference papers used to justify implementation and reporting choices.

## Files

- `advanced-ca-2025-2026-askisi1.pdf`: official assignment statement.
- `CAL-2024-geomean.pdf`: reference material for averages and speedup interpretation.
